# Security Policy

## Reporting Security Issues

DAPM is a documentation and methodology repository and does not contain production software or executable services.

If you identify:

* Sensitive information accidentally committed to the repository
* Security-related concerns involving repository configuration
* Malicious or inappropriate content
* Any issue that could impact repository integrity

please report it privately to the repository maintainer rather than opening a public issue.

---

## Responsible Disclosure

Please provide:

* A clear description of the issue.
* Steps to reproduce (if applicable).
* Any supporting evidence.

The issue will be reviewed as quickly as possible, and appropriate action will be taken.

---

Thank you for helping keep the DAPM repository secure and trustworthy.
